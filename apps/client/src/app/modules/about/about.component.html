About

This project is built using github, Angular, nx, scully and primer-css. It uses github for project management, ci/cd and hosting. Overall this project cn be considered a "JAMStack app", or really just a fancy static site that also uses Angular.

If you see any issues or just want to provide feedback, please create an issue here
